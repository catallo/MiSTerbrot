# MiSTerbrot v0.10.0

## Highlights
- Switched project release scheme to semantic versioning starting at **0.10.0**.
- Fixed startup random playlist behavior for auto-zoom target/theme sequencing.

## RNG / Playlist Fixes
- Target playlist width fixed from 4-bit to 5-bit (full 0..24 target space is now representable).
- Target playlist position counter widened to 5-bit (full 25-slot loop supported).
- Fisher-Yates termination fixed to include the final i=1 step (off-by-one removed).
- Shuffle candidate generation unified and widened using a 32-bit random pool.
- Entropy seed widened and mixed (now derived from full timestamp input path, not only 16-bit low slice).

## Theme Override Behavior
- OSD Theme now exposes **Auto** as explicit option.
- Theme override semantics clarified:
  - 0 = Auto (follow auto playlist/manual path)
  - 1..31 = fixed palette selection
